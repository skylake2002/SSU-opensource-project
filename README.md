# OpenPose를 활용한 원격 재활 운동 및 물리치료 도우미 시스템

***
# *작동 방식*
• 영상 통화를 통해 치료사의 시범 동작 전달

• 환자의 동작 수행

• 환자의 동작 분석 후 치료사의 동작과 비교

• 비교한 데이터를 기반으로 치료사의 피드백 제공

• 환자의 동작 수정

***
